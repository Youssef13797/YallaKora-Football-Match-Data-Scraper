# ⚽ YallaKora Football Match Data Scraper

## 📌 Overview
This project is a web scraping script built using Python to collect football match data from the YallaKora website.

It extracts daily match information including:
- League name
- Match teams
- Match result
- Match date

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Requests
- BeautifulSoup

---

## 📊 Dataset
The scraped data is saved as a CSV file:
- `yallakora_matches.csv`

---

## 🚀 Features
- Scrapes real-time match data
- Extracts structured data from HTML
- Saves clean data into CSV format
- Handles multiple leagues in one run
